# App Description

This proyect is a simple API created by an AI agent

Tech stack: FastAPI + simple storage JSON (memory.json).

## Example of user:

{
    "user_id": 1,
    "username": "anyname",
    "email": "anyone@anymail.com",
    "password": "savethequeen123"
    "age": 27,
    "languages": ["English", "Spanish"],
}

user_id is an autogenerated primary key.

## Project structure:

trying-ai-agents/
│
├── memory-bank/
│   ├── app-description.md
│   ├── implementation-plan.md
│   └── requirements.md
│
├── main.py          # ONLY app initialization, nothing else
├── src
    ├── __init__.py      # package initialization
│   ├── endpoints.py     # ALL route definitions (POST /users/, GET /users/)

│
├── .gitignore       # excludes .venv/ and memory.json
├── memory.json      # persistent storage (auto-created on startup)
├── requirements.txt # dependencies
└── .venv/           # virtual environment
